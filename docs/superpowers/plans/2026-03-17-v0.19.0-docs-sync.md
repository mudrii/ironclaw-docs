  ```markdown
  The built-in HTTP channel validates an HMAC-SHA256 signature sent in the `X-Webhook-Signature` request header (v0.19.0+, PR #970). The secret is **never** transmitted in the request body. The gateway computes `HMAC-SHA256(secret, body)` and compares using constant-time equality.

  ```
